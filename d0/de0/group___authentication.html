Detailed Description

User authentication, account management, and identity services

This module provides functionality for:

  • User login and authentication (email/password, device, platform)
  • OAuth2 token management and refresh
  • Multi-factor authentication (MFA)
  • Platform account linking (Steam, Epic, PlayStation, Xbox, etc.)
  • User verification and password reset
  • Legal agreements acceptance
  • Login queue management
  • GDPR compliance (data requests, account deletion)
